# Portfolio DAW

Portfolio personal desarrollado desde cero para practicar y consolidar los conocimientos adquiridos en 1º de Desarrollo de Aplicaciones Web (DAW).

## Objetivos

- Aplicar HTML5 semántico.
- Diseñar interfaces con CSS3.
- Practicar control de versiones con Git y GitHub.
- Crear un sitio web responsive.
- Desarrollar una estructura de proyecto profesional.

## Tecnologías

- HTML5
- CSS3
- Git
- GitHub
